# University Database Management System (SQL)

## Aim

To implement transaction management in a University Database using SQL by creating relational tables for Students, Courses, and Enrollments, performing transactions using **START TRANSACTION**, **SAVEPOINT**, **ROLLBACK**, and **COMMIT**, and retrieving data using **INNER JOIN**.

---

## Program Title

**Transaction Management and Savepoint Simulation in Student Enrollments**

## Tools Used

- MySQL
- MySQL Workbench
- SQL

---

## Concepts Covered

- Relational Database Design
- Primary Key
- Foreign Key
- DDL Commands
- DML Commands
- Transaction Control (START TRANSACTION, SAVEPOINT, ROLLBACK, COMMIT)
- INNER JOIN

---

## Database Tables

- **Student**
- **Course**
- **Enrollment**

---

## Features

- Creates relational database tables.
- Inserts sample student and course records.
- Demonstrates transaction management.
- Uses SAVEPOINT and ROLLBACK to handle errors.
- Retrieves data using INNER JOIN.
